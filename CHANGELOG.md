# Changelog

All notable changes to Frontier will be documented here.

## v0.1.0 — Initial Playable Prototype

### Added

* Project architecture
* GameManager autoload
* Survivor system
* Inventory system
* Data-driven ItemData resources
* Automatic ItemDatabase
* Search action
* Event log
* Gathering experience
* Knowledge system
* Inventory UI

### Notes

This version represents the first playable gameplay loop. Players can search for resources, gain experience, level up, and view their inventory through the interface.
