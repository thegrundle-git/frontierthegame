# Frontier Design Document

## Vision

Frontier is a hardcore survival RPG where every action permanently shapes your character, eventually growing from a lone survivor into a thriving civilization.

---

# Core Pillars

- Start with nothing.
- Discover knowledge naturally.
- Skills improve through use.
- Perks are earned permanently.
- Every era should feel like a new game.
- Death creates a legacy instead of ending the game.

---

# Progression

Personal Skills
- Lost on death.

Civilization Knowledge
- Persists forever.

Buildings
- Persist.

Items
- Can be recovered.

# Inventory Philosophy

All storage uses the same container system.

Examples:
- Survivor backpack
- Storage chest
- Animal nest
- Corpse inventory
- Trader inventory
- Settlement storage

Containers differ by:
- Weight capacity
- Slot capacity
- Allowed items
- Access rules
