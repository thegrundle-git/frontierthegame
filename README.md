# Frontier

Frontier is a text-driven survival and civilization-building game developed in Godot.

The game begins with a single survivor struggling to survive in the wilderness. Every action contributes to the growth of both the survivor and the world around them. Over time, simple resource gathering leads to crafting, camp building, settlement management, technological discovery, and eventually the development of a thriving civilization.

## Current Features

* Survivor system
* Inventory system
* Data-driven item database
* Search and gathering actions
* Gathering experience and leveling
* Knowledge progression
* Event log
* Dynamic inventory display

## Planned Features

* Multiple biomes
* Crafting system
* Camp construction
* Hunger and survival mechanics
* Settlements
* Population management
* Technology and discoveries
* Trade and economy
* Dynamic world events

## Built With

* Godot Engine
* GDScript

## Project Status

**Version:** v0.1.0

This is the first playable prototype. The core gameplay loop is functional, and future updates will expand the survival mechanics into a full civilization-building experience.
